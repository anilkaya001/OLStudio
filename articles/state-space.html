1  The General Framework

State-space models provide a unified framework for dynamic models with unobserved components. The system:

Transition
$$\boldsymbol{\alpha}_{t+1} = \mathbf{T}_t\boldsymbol{\alpha}_t + \mathbf{R}_t\boldsymbol{\eta}_t, \quad \boldsymbol{\eta}_t \sim \mathcal{N}(\mathbf{0}, \mathbf{Q}_t)$$
Measurement
$$\mathbf{y}_t = \mathbf{Z}_t\boldsymbol{\alpha}_t + \boldsymbol{\varepsilon}_t, \quad \boldsymbol{\varepsilon}_t \sim \mathcal{N}(\mathbf{0}, \mathbf{H}_t)$$

System matrices $\mathbf{T}_t, \mathbf{R}_t, \mathbf{Z}_t$ can be time-varying. This nests ARIMA, regression with time-varying coefficients, unobserved components, dynamic factor models, and many more.

2  Unobserved Components Model

The local linear trend model decomposes $y_t$ into trend $\mu_t$, slope $\beta_t$, and irregular $\varepsilon_t$:

$$y_t = \mu_t + \varepsilon_t, \quad \mu_{t+1} = \mu_t + \beta_t + \eta_t, \quad \beta_{t+1} = \beta_t + \zeta_t$$

State vector $\boldsymbol{\alpha}_t = (\mu_t, \beta_t)'$, transition $\mathbf{T} = \begin{pmatrix}1&1\\0&1\end{pmatrix}$, measurement $\mathbf{Z} = (1,0)$. Special cases: $\sigma_\zeta^2 = 0$ gives a local level with drift; $\sigma_\eta^2 = \sigma_\zeta^2 = 0$ gives a deterministic trend.

3  Time-Varying Parameters

TVP Regression
$$y_t = \mathbf{x}_t'\boldsymbol{\beta}_t + \varepsilon_t, \quad \boldsymbol{\beta}_{t+1} = \boldsymbol{\beta}_t + \boldsymbol{\eta}_t$$

Parameters follow random walks, capturing gradual structural change. The Kalman filter estimates $\boldsymbol{\beta}_t$ at each point, with the innovation variance $\mathbf{Q}$ governing the speed of parameter drift. When $\mathbf{Q} = \mathbf{0}$, this collapses to constant-parameter OLS.

4  Dynamic Factor Models

A large panel of $n$ variables is driven by $r \ll n$ common factors:

$$\mathbf{y}_t = \boldsymbol{\Lambda}\mathbf{f}_t + \boldsymbol{\varepsilon}_t, \quad \mathbf{f}_t = \boldsymbol{\Phi}\mathbf{f}_{t-1} + \boldsymbol{\eta}_t$$

$\boldsymbol{\Lambda}$ is the $n \times r$ loading matrix, $\mathbf{f}_t$ the $r \times 1$ factor vector. Estimation via Kalman filter MLE (small $n$) or principal components followed by Kalman smoothing (large $n$, Stock and Watson, 2002).

5  Missing Data and Mixed Frequencies

State-space models handle missing observations naturally: simply skip the update step when $y_t$ is unobserved. This enables mixed-frequency estimation (e.g., combining monthly and quarterly data), nowcasting (estimating current-quarter GDP from partial monthly indicators), and interpolation of irregular data.

6  Estimation of System Parameters

Unknown parameters in $\mathbf{T}, \mathbf{Z}, \mathbf{Q}, \mathbf{H}$ (collectively $\boldsymbol{\theta}$) are estimated by maximising the prediction error decomposition likelihood from the Kalman filter. The EM algorithm is an alternative: E-step runs the Kalman smoother; M-step updates $\boldsymbol{\theta}$ analytically. Bayesian estimation via MCMC (Gibbs sampling with data augmentation for the states) is increasingly standard.
