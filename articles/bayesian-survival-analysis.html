1  Objective

We model downside risk in BIST 100 (XU100.IS) as a survival problem: time until a large-loss event. A Bayesian proportional hazards setup provides posterior distributions for hazard and survival probability instead of single-point estimates.

2  Data from yfinance

Download daily close prices, compute log-returns, and define an event when return falls below a risk threshold (for example -2%). Non-events are right-censored observations. 

import numpy as np
import pandas as pd
import yfinance as yf

ticker = "XU100.IS"
px = yf.download(ticker, start="2010-01-01", auto_adjust=True, progress=False)["Close"].dropna()
r = np.log(px).diff().dropna()

threshold = -0.02
event = (r <= threshold).astype(int)

3  From Returns to Survival Data

Use spell lengths between severe-loss events as durations. If a spell is not terminated by an event before sample end, it is censored. 

durations, observed = [], []
spell = 0
for e in event.values:
    spell += 1
    if e == 1:
        durations.append(spell)
        observed.append(1)
        spell = 0
if spell > 0:
    durations.append(spell)
    observed.append(0)

df = pd.DataFrame({"time": durations, "event": observed})

4  Bayesian Proportional Hazards

With piecewise-constant baseline hazard, the survival function is:

$$S(t) = \exp\left(-\int_0^t \lambda(u)\,du\right), \qquad \lambda_i(t)=\lambda_0(t)\exp(x_i^\top\beta)$$

For market risk, covariates can include rolling volatility, drawdown state, or momentum regime. Posterior draws of $\beta$ quantify how these states alter loss-arrival risk.

5  PyMC Implementation Sketch

import pymc as pm
import pytensor.tensor as pt

interval_length = 5
bounds = np.arange(0, df.time.max() + interval_length + 1, interval_length)
n_intervals = len(bounds) - 1
n = len(df)
last = np.floor((df.time.values - 1e-2) / interval_length).astype(int)

death = np.zeros((n, n_intervals))
death[np.arange(n), last] = df.event.values

exposure = (df.time.values[:, None] >= bounds[:-1]).astype(float) * interval_length
exposure[np.arange(n), last] = df.time.values - bounds[last]

with pm.Model(coords={"intervals": np.arange(n_intervals)}) as model:
    lambda0 = pm.Gamma("lambda0", alpha=0.01, beta=0.01, dims="intervals")
    beta = pm.Normal("beta", mu=0.0, sigma=2.0)
    x = 0.0  # replace with covariate vector if available
    lam = pm.Deterministic("lambda_", pt.outer(pt.exp(beta * x), lambda0))
    mu = pm.Deterministic("mu", exposure * lam)
    pm.Poisson("obs", mu=mu, observed=death)
    idata = pm.sample(1000, tune=1000, target_accept=0.95, random_seed=42)

6  Risk and Survival Quantities

Posterior hazard gives the probability of a severe-loss event per interval; posterior survival gives the probability of no severe loss up to horizon $t$:

$$\Lambda(t)=\sum_{j \le t}\lambda_j\Delta_j,\qquad S(t)=\exp(-\Lambda(t))$$

Useful outputs are $P(T>20)$ (survival over 20 trading days), median survival time, and credible intervals for scenario-based risk reporting.

7  Extensions for XU100

Practical extensions include time-varying coefficients (random walk), volatility-state covariates, and separate models for crisis vs. calm regimes. These provide a Bayesian view of changing market fragility over time.
