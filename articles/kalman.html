1  State-Space Models

The Kalman filter (1960) provides the optimal recursive solution for linear Gaussian state-space models. The system consists of:

State Equation
$$\boldsymbol{\xi}_{t+1} = \mathbf{F}\boldsymbol{\xi}_t + \mathbf{R}\boldsymbol{\eta}_t, \quad \boldsymbol{\eta}_t \sim \mathcal{N}(\mathbf{0}, \mathbf{Q})$$
Observation Equation
$$\mathbf{y}_t = \mathbf{H}'\boldsymbol{\xi}_t + \boldsymbol{\varepsilon}_t, \quad \boldsymbol{\varepsilon}_t \sim \mathcal{N}(\mathbf{0}, \mathbf{R}_\varepsilon)$$

$\boldsymbol{\xi}_t$ is the unobserved state vector ($m \times 1$), $\mathbf{y}_t$ is the observed signal ($n \times 1$), $\mathbf{F}$ is the state transition matrix, and $\mathbf{H}$ maps states to observations.

2  Prediction Step

Given the filtered estimate $\hat{\boldsymbol{\xi}}_{t|t}$ and covariance $\mathbf{P}_{t|t}$:

State Prediction
$$\hat{\boldsymbol{\xi}}_{t+1|t} = \mathbf{F}\hat{\boldsymbol{\xi}}_{t|t}$$
Covariance Prediction
$$\mathbf{P}_{t+1|t} = \mathbf{F}\mathbf{P}_{t|t}\mathbf{F}' + \mathbf{R}\mathbf{Q}\mathbf{R}'$$

3  Update Step

When $\mathbf{y}_{t+1}$ is observed:

Innovation
$$\mathbf{v}_{t+1} = \mathbf{y}_{t+1} - \mathbf{H}'\hat{\boldsymbol{\xi}}_{t+1|t}$$
Kalman Gain
$$\mathbf{K}_{t+1} = \mathbf{P}_{t+1|t}\mathbf{H}(\mathbf{H}'\mathbf{P}_{t+1|t}\mathbf{H} + \mathbf{R}_\varepsilon)^{-1}$$
State Update
$$\hat{\boldsymbol{\xi}}_{t+1|t+1} = \hat{\boldsymbol{\xi}}_{t+1|t} + \mathbf{K}_{t+1}\mathbf{v}_{t+1}$$
Covariance Update
$$\mathbf{P}_{t+1|t+1} = (\mathbf{I} - \mathbf{K}_{t+1}\mathbf{H}')\mathbf{P}_{t+1|t}$$

4  Prediction Error Decomposition

The log-likelihood is computed from the innovation sequence:

Log-Likelihood
$$\ell = -\frac{Tn}{2}\ln(2\pi) - \frac{1}{2}\sum_{t=1}^{T}\left[\ln|\mathbf{F}_{t|t-1}| + \mathbf{v}_t'\mathbf{F}_{t|t-1}^{-1}\mathbf{v}_t\right]$$

where $\mathbf{F}_{t|t-1} = \mathbf{H}'\mathbf{P}_{t|t-1}\mathbf{H} + \mathbf{R}_\varepsilon$ is the innovation variance. This enables MLE of unknown system parameters.

5  Kalman Smoother

The Rauch-Tung-Striebel smoother provides estimates using the full sample $\{y_1, \ldots, y_T\}$:

Backward Recursion
$$\hat{\boldsymbol{\xi}}_{t|T} = \hat{\boldsymbol{\xi}}_{t|t} + \mathbf{J}_t(\hat{\boldsymbol{\xi}}_{t+1|T} - \hat{\boldsymbol{\xi}}_{t+1|t})$$

where $\mathbf{J}_t = \mathbf{P}_{t|t}\mathbf{F}'\mathbf{P}_{t+1|t}^{-1}$. The smoother gives lower variance than the filter: $\mathbf{P}_{t|T} \leq \mathbf{P}_{t|t}$ in the matrix sense.

6  Applications in Econometrics

The Kalman filter enables estimation of unobserved components models (trend-cycle decomposition), time-varying parameter regressions ($\boldsymbol{\beta}_t$ evolving as a random walk), ARMA models via state-space representation, dynamic factor models, and the estimation of potential output and the output gap. It is the workhorse of modern macroeconometrics.
